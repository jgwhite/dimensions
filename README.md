# Dimensions

Displays dimensions given by:

* `screen.width`
* `screen.height`
* `window.outerWidth`
* `window.outerHeight`
* `window.innerWidth`
* `window.innerHeight`

At the following phases of page load:

* script executed at the top of `<head>`
* script executed at the top of `<body>`
* on jQuery `ready` event
* on window `load` event
